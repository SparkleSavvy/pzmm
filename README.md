# pzmm
FOSS Mod manager for Project Zomboid
<img width="1484" height="1029" alt="image" src="https://github.com/user-attachments/assets/1fb254dd-edb9-4019-be69-d01ed1149dcb" />

# Features
- Download multiple mods
- Update mods
- Remove selected or all mods
- View installed mods and dependencies
- Configure directories
- Miscellaneous quality-of-life features

# How to use
> [!NOTE]
> You must have installed python, steamcmd!
1. Install depencies
```
pip install PyQt6 PyQt6-WebEngine requests
```
3. Run script
```
py pzmm.py
```
